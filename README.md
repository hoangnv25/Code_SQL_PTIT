# Bài Giải SQL PTIT (MySQL)

Đây là phần code lời giải cho các bài tập SQL trên trang luyện tập của PTIT:  
**Link làm bài:** https://db.ptit.edu.vn/

Ngôn ngữ sử dụng: **MySQL**

Hiện tại tôi **chưa hoàn thành toàn bộ**, mới chỉ giải được một phần số lượng bài và sẽ tiếp tục cập nhật thêm trong tương lai.

**Cập nhật mới nhất:** *10/12/2025*

---

## Ghi chú

- Các bài được đặt tên theo mã số thứ tự trên hệ thống của PTIT.  
- Một số bài có thể có nhiều cách giải; tôi chọn phương án AC :v  
- Nếu PTIT cập nhật đề hoặc dữ liệu, lời giải có thể cần điều chỉnh lại.
